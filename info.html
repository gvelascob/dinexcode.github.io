Ticket assignment automation for Zendesk

Automated even load distribution = happy support agents = happy supervisors :)

(Zendesk community forum)

Round Robin App is a service that automatically assigns tickets to the agents in your Zendesk® in a round robin fashion.

Features:
  • Round robin assignment
  • Agent ticket queue limiting
  • Queue prioritization
  • Agent skills based ticket assignment

Coming soon:
  • Assigning by agent's work schedules (support teams in different time zones)
  • Assigning tickets to available agents
  • Assigning a ticket by agent request
